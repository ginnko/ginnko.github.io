不同方式创建对象以及导致的原型链

  1. 使用字面两创建

    • var o = {a: 1};, 原型链：o ---> Object.prototype ---> null。
    • var b = ['yo', 'whadup', '?'];, 原型链： b ---> Array.prototype ---> Object.prototype ---> null。
    • function f() {return 2}, 原型链： f ---> Function.prototype ---> Object.prototype ---> null。
