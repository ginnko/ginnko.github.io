  1. 代理对象

    let p = new Proxy(target, handler);

    • target表示用Proxy包装的目标对象（可以是任何类型的对象，包括原生数组，函数，甚至另一个代理）。

    • handler表示一个对象，其属性是当执行一个操作时定义代理的行为的函数。（handler的属性键都是对象的原生函数？感觉是哦）

    2. 无操作转发代理

      使用一个原生的js对象，代理会将所有应用到它的操作转发到这个对象上。
